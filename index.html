About WeatherWatch

WeatherWatch is a private automation tool designed to help publish weather advisories, public updates, and related information to authorized social media pages.

Purpose

The app helps reduce manual posting work by generating and publishing approved weather-related content through official Page access granted by the Page administrator.

Data Use

WeatherWatch only uses permissions necessary to identify authorized Pages and publish approved posts. It does not sell user data or provide public login services.
